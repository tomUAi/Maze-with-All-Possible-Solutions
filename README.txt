- This app can give you all the possible solutions of a maze.

- It need a input of a matrix with 0 and 1, where 0 are the paths and 1 are the walls. 

- The matrix need to have just one entry and one exit.

- The entry is on the left and the exit is on right.

- App_v1 and App_v2 are basicaly the same but App_v1 is more optimied.
